# Dockerfiroonga

Dockerfile generator for Groonga family.

## Installation

    $ gem install dockerfiroonga

## Usage

    $ dockerfiroonga PLATFORM [Xroonga]

### PLATFORM

* `debian[:wheezy]` (APT)
* `debian:XXX` (.tar.gz)
* `ubuntu` (PPA)
* `centos` (yum)

### Xroonga

* `groonga` (default)
* `rroonga`
